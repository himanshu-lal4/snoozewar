## KEY THEMES FOR SNOOZEWAR COMPETITIVE POSITIONING

1. **Subscription rage is real** — Users feel trapped, can't cancel, get price-hiked. SnoozeWar's transparent 7-usage trial + clear pricing is a differentiator.

2. **Tracking accuracy is the #1 complaint** — Sleep Cycle can't reliably detect awake vs asleep with just accelerometer/mic. SnoozeWar doesn't pretend to be a sleep tracker; it's an enforcement tool.

3. **Alarm reliability is table stakes** — Multiple users report Sleep Cycle alarms not going off. SnoozeWar's alarm MUST be bulletproof (AlarmKit on iOS 26).

4. **Data loss destroys trust** — Years of data vanishing is devastating. SnoozeWar should have robust backup from day 1.

5. **Partner/shared bed is unsolved** — Sleep Cycle struggles in shared beds. Not SnoozeWar's problem (we don't track sleep movement).

6. **Orthosomnia is a real risk** — Sleep tracking can cause anxiety. SnoozeWar's "Life Score" should be framed as behavioral, not clinical.

7. **Battery drain is a dealbreaker** — 30-60% overnight is unacceptable. SnoozeWar only needs to be active for 30 min post-wake, not all night.

8. **"Used to be great" sentiment** — Long-term users feel the app has degraded. Opportunity to capture churned Sleep Cycle users.

9. **Customer support is nearly nonexistent** — Users can't reach anyone. Even basic responsiveness would be a competitive advantage.

10. **Free version feels gutted** — "Everything else is premium" frustration. SnoozeWar's 7-usage full-experience trial avoids this.
